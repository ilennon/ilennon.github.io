range

Definition

A new array type that can hold string, int and double data types.

Usage


int size(); Returns total size of range.
int size_str(); Returns string size of range.
int size_int(); Returns int size of range.
int size_double(); Returns double size of range.

bool in_array(data); Returns if data is exist in range.

void add(data, bool unique = false); Adds a new data to range.

vector<string> all(); Returns all datas as string array.
string all(bool str); Returns all datas as string. Bool value doesn't matter.

vector<string> all_types(); Returns all data types as string array.
string all_types(bool str); Returns all data types as string. Bool value doesn't matter.

vector<string> all_str(); Returns all string values as string array.
string all_str(bool str); Returns all string values as string. Bool value doesn't matter.

vector<int> all_int(); Returns all int values as int array.
string all_int(bool str); Returns all int values as string. Bool value doesn't matter.

vector<double> all_double(); Returns all double values as double array.
string all_double(bool str); Returns all double values as string. Bool value doesn't matter.

string at(unsigned int index); Returns nth value as string.
string at_str(unsigned int index); Returns nth string value as string.
int at_int(unsigned int index); Returns nth int value as int.
double at_double(unsigned int index); Returns nth double value as double.

void set(unsigned int index, var); Sets nth value as defined. Note: If data type is different, nothing will change. At least for now.

void set_str(unsigned int index, string var); Sets nth string value as defined.
void set_int(unsigned int index, int var); Sets nth int value as defined.
void set_double(unsigned int index, double var); Sets nth double value as defined.

void truncate(); Sets all double and int values to 0 and string values to "".

void clear(); Deletes everything.
